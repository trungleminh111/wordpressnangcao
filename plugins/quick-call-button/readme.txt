== Description ==

✅ It's easy to add a quick call button on the website.
✅ Quick Call Button is a plugin add quick call button that helps your customers avoid typing in phone numbers that can be instantly dialed through your website.
✅ Increase the number of converting customers to your and sales service.
✅ Quick Call Button is a powerful development tool that gives you all the options from button placement to custom colors from the administration page.
Show phone numbers or any text on any screen.

Click Tracking Google Universal Analytics.

== Installation ==

1. Upload the entire 'quick-call-button' folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Use the Settings->Quick Call Button screen to configure the plugin.

== Frequently Asked Questions ==

= How does the quick call button work ? =

You need to install and activate the quick call button plugin and set up the necessary phone number on the settings page of the quick call button plugin.

= How to make quick call buttons available on desktop as well ? =

You need to go to the settings page of the quick call button, At the top you will see ( Display On Screen ) you need to set a number larger than the default number (2000) for example.
